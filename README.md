This project creates a program that outputs Leminski phrases randomly.

## Development 
To build, run the following command:

- `gcc frases.c -o frases.exe` , on Windows,
- `gcc frases.c -o frases.out` , on Linux.

To run:

- `.\frases.exe`, on Windowns,
- `./frases.out`, on Linux.
